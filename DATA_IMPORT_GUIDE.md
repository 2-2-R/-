# 数据导入使用说明

## 概述

[`import_data.py`](SmartTrainingScheme/backend/core/management/commands/import_data.py) 是一个 Django 管理命令，用于导入物联网工程专业数据，支持 OBE 四级映射模型的多表级联导入。

## 功能特性

### 1. 多表级联导入

脚本按照以下顺序导入数据，确保外键关系正确：

1. **专业信息（Major）**：从 `major` 工作表读取专业名称和代码
2. **毕业要求（GraduationRequirement）**：从 `graduation_requirement` 工作表读取，自动关联到专业
3. **课程信息（Course）**：从 `course` 工作表读取，自动关联到专业

### 2. 智能关联逻辑

- **自动提取指标点**：如果毕业要求内容包含分号（`;`、`；`）或句号（`。`）分隔的多个要点，脚本会自动创建 [`IndicatorPoint`](SmartTrainingScheme/backend/core/models.py:76) 实例
- **权重自动分配**：为每个指标点自动计算平均权重，确保总和为 1.0
- **外键自动关联**：确保所有外键关系正确建立

### 3. 健壮性设计

- **垂直格式解析**：支持"模型-字段-值"的垂直数据格式
- **容错处理**：
  - 处理空值（NaN）
  - 处理学分为空的情况
  - 自动去除字符串首尾空格
- **事务保护**：使用 Django 事务确保数据一致性，导入失败时自动回滚

### 4. 详细反馈

- 显示每个导入步骤的进度
- 标记创建或更新操作
- 显示最终统计信息（专业数、毕业要求数、课程数）
- 错误时显示详细的堆栈跟踪

## 使用方法

### 基本用法

```bash
cd SmartTrainingScheme/backend
python manage.py import_data --file ../../物联网工程专业.xlsx
```

### 清空现有数据后导入

```bash
python manage.py import_data --file ../../物联网工程专业.xlsx --clear
```

### 参数说明

- `--file`：Excel 文件路径（默认：`物联网工程专业.xlsx`）
- `--clear`：导入前清空现有数据（可选）

## Excel 文件格式要求

### 工作表结构

Excel 文件应包含以下工作表：

1. **major**：专业信息
2. **graduation_requirement**：毕业要求
3. **course**：课程信息

### 数据格式

数据采用垂直格式（模型-字段-值）：

| Column1 | Column2 | Column3 |
|---------|---------|---------|
| 模型    | 字段    | 值      |
| Major   | name    | 物联网工程 |
| Major   | code    | 080905  |

### 字段说明

#### Major（专业）
- `name`：专业名称
- `code`：专业代码（唯一）

#### GraduationRequirement（毕业要求）
- `major`：所属专业名称
- `sequence`：序号
- `content`：要求内容

#### Course（课程）
- `code`：课程代码（唯一）
- `name`：课程名称
- `credits`：学分
- `major`：所属专业名称

## 数据模型关系

```
Major（专业）
  ├── GraduationRequirement（毕业要求）
  │     └── IndicatorPoint（指标点）*自动创建
  └── Course（课程）
```

## 注意事项

1. **数据库迁移**：首次使用前，请确保已运行数据库迁移：
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

2. **编码问题**：脚本已添加 UTF-8 编码声明，支持中文内容

3. **数据验证**：导入后建议检查数据库，确认数据正确性

4. **备份建议**：使用 `--clear` 参数前，建议备份现有数据
